# Hatchet (hatchet)

Hatchet is a distributed task queue and workflow engine for building resilient backend applications with durable execution. It provides AI agents, durable workflows, and background tasks with first-class support for Python, TypeScript, Go, and Ruby. Available as a managed cloud service or self-hosted under the MIT license.
